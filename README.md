# Dot-Files

## Quick Start

1. **Clone the repository:**
   ```bash
   git clone  https://github.com/Teeheene/Dot-Files.git ~/.Dot-Files
   cd ~/.dotfiles
   ```

2. **Install prerequisites** 

3. **Use stow to create symbolic links:**
   ```bash
   stow nvim 
   ```

4. **First-time setup:**
   ```bash  
   # Open Neovim to auto-install plugins
   nvim
   ```
